## Let's create a website using NextJS and TailwindCSS which satisfies the following acceptance criteria:
- Main theme is local meetup named: "AI Jam Łódź"
- nextjs+sqlite+tailwind is the main technology used
- meetups historical log
- about page with history of the meetup
- localization (plain markdown website to be edited or fetched from elsewhere)
- next meetup details
- QR code to a page with basic info
- starter pack page with info about the meetup for new members
